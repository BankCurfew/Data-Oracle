# Real agent LINE chat style beats bot formatting — data proves it. Analyzed 5,544

Real agent LINE chat style beats bot formatting — data proves it. Analyzed 5,544 agent messages: median 50 chars (bot was 300+), 99.2% no line breaks, 0% bullet lists, 32% multi-message sequences. Winning agents use real numbers ("เบี้ย 19,853/ปี") not general principles, ask before answering, and break into 2-3 short messages. Training data formatted with markdown/bullets creates bots that sound like documentation. Rewrite system prompt: 50-80 char target, no bullets, no \n, ask first.
