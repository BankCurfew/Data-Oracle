# Day closure retro prevents cold boot — 15-min evening recap = full awareness next morning

Even a brief end-of-day session (recap + retro) means the next morning starts with full context instead of cold-booting from git log. The retro crystallizes what happened, the recap surfaces pending work, and the handoff file gives the next session a running start. Cost: 15 minutes. Payoff: 10+ minutes saved on next session orientation.
